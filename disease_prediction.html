Disease Prediction System

AI & Rule-Based Inference

An intelligent web-based application that predicts possible diseases based on user-provided symptoms. The system utilizes logic programming and a full-stack workflow to streamline patient-doctor interactions.

Technologies Used

  • Prolog (AI Logic)
  • PHP & MySQL
  • HTML, CSS, JS

Key Features

  • Symptom-based Reasoning
  • Secure User Dashboards
  • Doctor Appointment Booking
  • Patient History Tracking
