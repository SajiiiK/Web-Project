Picture Gallery Website

Responsive Web Design Project

The Picture Gallery Website is a simple and responsive web project created using HTML and CSS. It displays images in a clean grid layout with a focus on visual presentation and user experience.

Technologies Used

  • HTML5
  • CSS3

Key Features

  • Responsive image grid layout
  • Clean and minimal design
  • Hover effects for images
  • Optimized for different screen sizes
